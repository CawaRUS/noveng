# NOVENG (Novel Engine) v0.2 

Легковесный движок для текстовых новелл на **C++**, разработанный с упором на производительность и простоту написания сценариев. 

> **Разработчик:** TheCawa (CawaRUS)  
> **Стек:** C++17, miniaudio, nlohmann/json

---

##  Особенности
*  **Сценарный движок:** Написание игры в обычном `.txt` файле.
*  **Аудио:** Поддержка MP3/WAV (фоновая музыка, цикличные треки, перемотка)
*  **Система сохранений:** Сериализация состояния в JSON (позиция, цвета, музыка)
*  **Текстовый UI:** Поддержка ANSI-цветов для персонажей и эффектов.
*  **Настройки:** Громкость и скорость печати текста сохраняются между сессиями.

---

##  Структура сценария
Сценарии хранятся в `res/scenario/`. Движок поддерживает три типа строк:

1. **Имя персонажа:** `[Имя]` - задает текущего говорящего.
2. **Текст:** Обычная строка текста для вывода с эффектом печати.
3. **Команды:** Пишутся в фигурных скобках `{команда:аргументы}`.

### Доступные команды:

**Читать в *documentation.md***

---

##  Управление
* **Enter:** Пропустить анимацию текста / Следующий диалог.
* **S (ы):** Быстрое сохранение в Слот 1.
* **Стрелки / Enter:** Навигация в меню и настройках.
* **ESC:** Выход из настроек с сохранением.

---

##  Сборка
Проект использует простой скрипт сборки. 
Зависимости:
* `miniaudio.h` (в комплекте)
* `json.hpp` (nlohmann/json)
* Компилятор с поддержкой C++17 (GCC/MSVC)
